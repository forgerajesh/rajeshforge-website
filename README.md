# Rajesh Forge Website

Production-ready static website for **product.rajeshforge.com**.

## Pages
- `/` Home
- `/pages/about.html`
- `/pages/services.html`
- `/pages/case-studies.html`
- `/pages/insights.html`
- `/pages/contact.html`
- `/pages/privacy.html`
- `/pages/terms.html`

## Run locally
```bash
cd projects/rajeshforge-website
python3 -m http.server 8080
# open http://localhost:8080
```

## Deploy
Deploy this folder to Netlify, Vercel (static), GitHub Pages, or any Nginx host.

## Final setup before go-live
- Replace `info@rajeshforge.com`
- Add real LinkedIn URL
- Add real Calendly booking URL
- Add favicon/logo
- Point DNS for `product.rajeshforge.com` to your hosting provider
